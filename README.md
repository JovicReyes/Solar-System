## Dependencies
- Numpy
- Packaging
- Plotly
- Tenacity

## How to use it

1. Install dependencies: ``pip install -r requirements.txt``
2. Activate virtual enviroment (macOS): ``source venv/bin/activate`` 
3. Run: ``Python script.py``
